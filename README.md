# Learning rust

Workspace to practice rust with the Rust Programming Language book as support

## Getting-started

Install rust:

```sh
curl https://sh.rustup.rs -sSf | sh -s -- -y
```

Install a linker:

```sh
apt install -y gcc
```

Restart the terminal and check the installation:

```sh
rustc --version
```

To update rust:

```sh
rustup update
```

To uninstall rust:

```sh
rustup self uninstall
```
